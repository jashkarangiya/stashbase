# Overview

StashBase turns local files into Agent-ready context.

It makes Markdown, HTML, PDF, DOCX, images, and other local files previewable, editable where appropriate, searchable, and available to AI agents through MCP.

Core value:

- Make local files stable context that agents can read and search.
- Keep the product file-first and local-first.
- Avoid creating another closed knowledge base or binding context to one agent.
- Keep the product surface centered on workspace, Agent Panel, and document indexing.
